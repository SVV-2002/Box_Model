BOX  MODEL

The CSS box model is essentially a box that wraps around every HTML element. It talks about design and layout. It consists of: margins, borders, padding, and the actual content.

Content - It is the area where text and images appear.

Padding - It clears an area around the content. It is transparent.

Border - It is a border around the padding and content.

Margin - It clears an area outside the border. It is transparent.
